# Evolution 1 Sample Data

This is a set of pre-seed data for EV1. There are 2 included files

## books.csv

This is a set of 12 real Sci-Fi and Fantasy books with matching ISBN data and 50% royalty rates (except for 2). There is a header line

## records.csv
This is a set of 25 sales records. Each record is an ISBN, a month/year date of record, a random sales quantity between 1 and 20, a total revenue, a royalty total, and a y/n for paid status. This file also includes a header line
